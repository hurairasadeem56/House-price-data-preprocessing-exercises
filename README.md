# House Price Data Preprocessing Exercises

This repository contains a beginner-friendly Jupyter Notebook for learning data preprocessing using a house price dataset.

## Topics Covered
- Loading CSV data with Pandas
- Exploring dataset shape, data types, and summary statistics
- Checking and handling missing values
- Removing duplicate records
- Dropping unnecessary columns
- Encoding categorical variables
- Preparing data for machine learning

## Files
- `House_Price_Data_Preprocessing_Learning_Notebook.ipynb` — main learning notebook
- `HousePricePrediction.csv` — dataset file required to run the notebook

## Requirements
```bash
pip install pandas jupyter
How to Run
Clone this repository.
Place HousePricePrediction.csv in the same folder as the notebook.
Open the notebook using Jupyter Notebook or JupyterLab.
Run all cells step by step.
Learning Goal
The goal of this repository is to help beginners understand basic data preprocessing concepts before applying machine learning models.
